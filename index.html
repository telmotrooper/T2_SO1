What is Nanvix?

Nanvix is a Unix like operating system created by Pedro H. Penna to address the demands of emerging manycore processors. It targets cluster-based architectures that rely on distributed and shared memory, and it was designed from scratch to deliver cutting edge performance, while enabling backward compatibility with existing software.

Current Features

  • Time-Sharing
  • Multiprogramming
  • Virtual Memory
  • POSIX Conformance
Upcoming Features
  • Multithreading
  • OpenMP Support
  • Remote File System
  • MPI Support
